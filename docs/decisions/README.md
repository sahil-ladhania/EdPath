# docs/decisions

**The canonical product/design decisions for v1 are locked in [`../reference/design-decisions.md`](../reference/design-decisions.md)** (C1–C6, D1–D24, numeric bounds B1–B9). Read that first — do not restate or re-decide it here.

This folder is for **new** Architecture Decision Records made **during the build** — decisions that arise after the v1 lock and aren't already covered by `design-decisions.md`.

- Use [`ADR-template.md`](./ADR-template.md) as the starting point for each new ADR.
- One file per decision: `ADR-NNNN-short-title.md`.
- Record context, the decision, alternatives considered, and tradeoffs — the same shape as the locked decisions.

If a new decision changes something already locked in `design-decisions.md`, **flag the conflict in the ADR** — don't silently edit the locked doc.
